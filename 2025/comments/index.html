We got comments!

Long live freedom of speech!

Posted by Ivo on Thursday, September 18, 2025

Introducing Twikoo

What bolg would this be if it did not have any comment section? A shitty one, that’s what. So one of the first things I’ve worked on after officially publishing this bolg was figure out how to add comments. Guess I succeeded.

Wow, Ivo, that’s so cool! How did you do it?

I’m glad you asked, fellow user. You see, this bolg is powered by Hugo (credited below). It’s a cool framework that just… builds sites for you! Easy to install, especially if you’re on Linux, and it needs few commands to get things done right. It also supports themes. Those are custom layouts built by hardworking people so that lazy folks like me can just take them for their own websites. (This one site’s theme is also credited below.) Well, it seems that this theme integrates the front-end of two comment systems. From what I’ve understood, one of them is paid. As the genius Linus Torvalds once said:

Software is like sex: It’s better when it’s free.

Because of my mentality, we got stuck with the second option. Twikoo. Click on that link and you’ll see their website is in Chinese. There is an option to switch to English, but it barely does anything. The documentation so far is entirely in Mandarin Chinese. My will, however, is unyielding. The language barrier did nothing to stop me in integrating the back-end in this great bolg. Google Translate helped a lot…

Honestly, it’s a straightforward process. There are several ways to deploy the back-end. Twikoo offers an extensive list with all the different services which can do so. I chose Vercel. It can help you fork GitHub repositories, and I barely needed to meddle with the settings. A database is also needed to store all the posts, users and whatnot. For that the peoples at Twikoo made me use MongoDB.

Let there be comments!

Yeah, that’s about it, my fellow 0.75 readers. Go bananas and comment away! There will be no censorship on the administrators’ behalf, except for attacks to my person. like and subscribe and hit the notifications bell for more content. Hmm, I should add an emailing system next to spam potential users whenever new posts come out…

Toodles.
